# Isaac Diagrams

This directory contains diagrams illustrating NVIDIA Isaac concepts and architecture for the Physical AI & Humanoid Robotics textbook.

## Diagram Categories

### Architecture Diagrams
- Isaac ROS architecture overview
- GPU-accelerated perception pipeline
- Isaac navigation system architecture

### Component Diagrams
- Isaac ROS message types and interfaces
- Hardware-accelerated computing modules
- Sensor integration patterns

### Flow Diagrams
- Perception processing pipeline
- Navigation and path planning flow
- Real-time control systems

### Integration Diagrams
- Isaac-ROS integration patterns
- GPU-CPU task distribution
- Multi-sensor fusion architectures

## File Naming Convention

Diagram files follow the pattern: `[topic]_[concept].[format]`
- `isaac_architecture.png` - Isaac system architecture overview
- `gpu_acceleration_flowchart.png` - GPU acceleration pipeline
- `navigation_system.png` - Isaac navigation architecture
