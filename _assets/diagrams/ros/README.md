# ROS Diagrams

This directory contains diagrams illustrating ROS 2 concepts and architecture for the Physical AI & Humanoid Robotics textbook.

## Diagram Categories

### Architecture Diagrams
- ROS 2 architecture overview
- Node-executor-container relationship
- Communication patterns (publisher-subscriber, service-client)

### Network Diagrams
- Multi-robot communication
- DDS communication patterns
- Network configuration examples

### Component Diagrams
- Node structure and lifecycle
- Message and service definitions
- Action architecture

### Flow Diagrams
- Communication flow between nodes
- Data processing pipelines
- Lifecycle management flows

## File Naming Convention

Diagram files follow the pattern: `[topic]_[concept].[format]`
- `architecture_overview.png` - ROS 2 architecture overview
- `node_communication_flowchart.png` - Communication patterns
- `lifecycle_flow.png` - Node lifecycle diagram
