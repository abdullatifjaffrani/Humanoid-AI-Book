# Simulation Diagrams

This directory contains diagrams illustrating simulation concepts and systems for the Physical AI & Humanoid Robotics textbook.

## Diagram Categories

### Architecture Diagrams
- Gazebo simulation architecture
- Simulation-ROS integration patterns
- Physics engine integration

### Environment Diagrams
- World file structure and components
- Model composition and hierarchies
- Sensor placement and configuration

### Component Diagrams
- URDF/SDF model structure
- Plugin architecture
- Sensor simulation models

### Flow Diagrams
- Simulation lifecycle
- Real-time vs. non-real-time simulation
- Multi-robot simulation coordination

## File Naming Convention

Diagram files follow the pattern: `[topic]_[concept].[format]`
- `gazebo_architecture.png` - Gazebo architecture overview
- `urdf_structure_flowchart.png` - URDF model structure
- `sensor_simulation.png` - Sensor simulation diagram
