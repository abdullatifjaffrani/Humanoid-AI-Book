# Quick Start Guide

Get started with the Physical AI & Humanoid Robotics Textbook in minutes.

## Overview

This comprehensive textbook guides you through a 14-week curriculum covering:
- ROS 2 Foundations
- Gazebo/Unity Simulation
- NVIDIA Isaac Platform
- Vision-Language-Action Systems
- Capstone Autonomous Humanoid Robot Project

## For Students

### Getting Started
1. Begin with **Module 1: ROS 2 Foundations** (Weeks 1-3)
2. Progress sequentially through each module
3. Complete all hands-on labs and exercises
4. Work through the capstone project in Week 14

### Learning Path
- **Weeks 1-3**: Master ROS 2 fundamentals
- **Weeks 4-5**: Learn simulation environments
- **Weeks 6-7**: Explore NVIDIA Isaac platform
- **Weeks 8-14**: Advanced VLA systems and capstone project

## For Instructors

### Course Setup
1. Review the complete curriculum structure
2. Set up required software environments (ROS 2, Gazebo, Isaac)
3. Prepare lab environments for students
4. Plan for 14-week semester schedule

### Resources
- Learning objectives for each week
- Lab exercises with solutions
- Assessment rubrics
- Presentation materials

## For Lab Engineers

### Environment Setup
1. Install ROS 2 Humble Hawksbill
2. Set up Gazebo simulation environment
3. Configure NVIDIA Isaac Platform (if available)
4. Prepare hardware platforms for practical exercises

## Essential Resources

### Software Requirements
- Ubuntu 22.04 LTS (recommended)
- ROS 2 Humble Hawksbill
- Gazebo Garden or newer
- NVIDIA Isaac ROS (if GPU hardware available)

### Hardware Requirements (Optional)
- NVIDIA Jetson platform for Isaac examples
- Robot hardware for practical exercises
- RGB-D camera for vision exercises
